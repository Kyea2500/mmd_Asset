3Dモデル　MMD用PMX
ういまま 通常衣装 1.0

Vtuberのういままファンメイドモデルです。


絆創膏、ニプレス関連モーフは眉モーフの所に入っています。
靴はその他モーフの一番下ON靴モーフで出てきます。


※ういままの二次創作ガイドラインを優先して従ってください。
