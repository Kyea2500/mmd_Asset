◇◇概要◇◇

========================================================================================================================================================================


この作品はプロジェクトセカイ！feat.初音ミクの登場キャラクターである暁山瑞希のファンモデルです。

なお、本モデルの衣装・体はVroidベースで作成しております。

また本モデル変換に様々なプラグインを使用しております。

それらの製作者様及び関係者の皆様にこの場を借りて感謝申し上げます。

◇◇モデルの仕様◇◇

========================================================================================================================================================================

・UV・トポロジーについて

　あまりきちんと整理していません。UVはオーバーラップ・ミラー展開したままのものが多々あります。ご了承ください。


・MMDボーンについて

　本モデルはWinglayer様制作のVRM2PmxConverterを利用してPmx化しています。自動変換のため不要なボーンが設定されている可能性があります。


・MMDの揺れものについて

　あまり調整をしておりません。より良い設定があるとおもいますのでお好みで変更してください。


・スカートの貫通

　大きく動くと貫通します。プラグインの使い方が分からないので手動で設定しましたが無理でした。。。
　こうするといいなどアドバイスがあれば是非お願いします。。。
